Bachelor Project

Algorithms with Albot /'ælbɒt/ is an interactive visualization tool for computer science algorithms. It features: Story mode: Guided exploration of Binary Search Trees, covering balance, running time, and complexity. Sandbox mode: Free experimentation with algorithms. Step-by-step animations and controllable inputs for in-depth learning. Practical application for solving exercises and understanding algorithmic concepts

Technologies: HTML, CSS, JavaScript, Threejs
